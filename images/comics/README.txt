PNG to JPG Converter v2

How to use:
1. Unzip this folder.
2. Double-click run_png_to_jpg.bat.
3. Paste the folder path that contains PNG images.
   - Paths copied from Windows Explorer with quotes are supported.
   - Dragging a folder onto the BAT file is also supported.
4. Choose whether to overwrite existing JPG files.

Common fix:
If you see "Folder does not exist", check:
- The path does not contain a typo.
- The folder really exists.
- The path was not copied with an unmatched quote.
- Example:
  D:\your_folder\images\comics
